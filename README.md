## plusコマンド
標準入力から読み込んだ数字の合計値を出すプログラムです。

$ seq 数 | ./plus

を行うと、 

数+(数-1)+(数-2)・・・・・+2+1

の計算ができます。

### 例

$ seq 5 | ./plus
15


↑(5+4+3+2+1=15)
